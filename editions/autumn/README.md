# Autumn Pamphlet — Web Edition

Web version of the **Autumn companion edition**: *"Human in the Loop: What You Think It Means vs. What It Actually Is"*.

A pamphlet for annotation teams correcting six common misconceptions about Human-in-the-Loop ML systems.

## Files

| File | Description |
|------|-------------|
| `index.html` | Self-contained editorial web experience (Cormorant Garamond, parchment palette). Open directly in any browser — no server required. |
| `source.md` | Markdown source for the pamphlet content. |

## Usage

Open `index.html` in any modern browser. The file is fully self-contained (no external dependencies).

To rebuild the PDF from source, use the root `Makefile` target `autumn-pdf`.
