# Ανθρωποκεντρική ΜΜ στην Υγεία και την Επιστήμη

Η υγειονομική περίθαλψη και η επιστήμη αντιπροσωπεύουν δύο από τους τομείς όπου η HITL ML είναι πιο συνεπαγωγική και πιο αμφιλεγόμενη. Τα διακυβεύματα είναι υψηλά: μια χαμένη διάγνωση καρκίνου ή ένας ελαττωματικός φαρμακευτικός στόχος έχει πραγματικό ανθρώπινο κόστος. Ο σχολιασμός απαιτεί σπάνια και ακριβή εμπειρογνωμοσύνη. Οι ρυθμιστικές απαιτήσεις περιορίζουν αυτό που μπορούν να κάνουν τα μοντέλα και πώς πρέπει να επικυρώνονται. Και σε αντίθεση με την ΕΦΓ, όπου το πρόβλημα σχολιασμού είναι εν μέρει κοινωνικής κατασκευής, εδώ υπάρχει συχνά γνήσια αλήθεια εδάφους — ένας όγκος παρευρίσκεται ή δεν παρευρίσκεται — ακόμα και αν κανένας μεμονωμένος παρατηρητής δεν μπορεί να το καθορίσει αξιόπιστα.

Το κυρίαρχο πλαίσιο στη λαϊκή κάλυψη είναι «ΤΝ vs. άνθρωπος»: θα αντικαταστήσει η ΤΝ τους ακτινολόγους; Αυτό το πλαίσιο είναι λανθασμένο με τρόπο που έχει σημασία. Το πραγματικό ερώτημα είναι τι μορφή ανθρώπου-ΤΝ συνεργασίας παράγει καλύτερα αποτελέσματα από οποιoδήποτε μόνο του, και πώς να χτίσουμε συστήματα που επιτρέπουν αυτή τη συνεργασία αντί να τη διαταράσσουν.

## Ανάλυση Ιατρικής Εικόνας

Η ιατρική απεικόνιση — ακτινολογία (ακτινογραφία, CT, MRI), παθολογία (ιστολογικές τομές), δερματολογία, οφθαλμολογία — είναι ο τομέας όπου η ιατρική ΤΝ έχει προχωρήσει ταχύτατα.

### Απαιτήσεις Εμπειρογνώμονα Σχολιαστή

Ο σχολιασμός ιατρικής εικόνας απαιτεί συνήθως ιατρούς με συγκεκριμένη υποειδικότητα εκπαίδευση. Αυτό κάνει τον σχολιασμό:

- **Αργό:** Οι ειδικοί έχουν περιορισμένο χρόνο· ο σχολιασμός ανταγωνίζεται με κλινικά καθήκοντα
- **Ακριβό:** Τα κόστη κυμαίνονται από δεκάδες έως εκατοντάδες δολάρια ανά σχολιασμένη περίπτωση, ανάλογα με υποειδικότητα, αρθρωτικότητα και πολυπλοκότητα έργου
- **Μεταβλητό:** Ακόμα και ειδικοί διαφωνούν, ειδικά για οριακές περιπτώσεις — γεγονός που συχνά αντιμετωπίζεται ως πρόβλημα αλλά είναι πραγματικά ενημερωτικό

### Διαρθρωτική Μεταβλητότητα Ακτινολόγων

Η μεταβλητότητα αναγνωστών είναι καλά τεκμηριωμένη στην ακτινολογία. Για ερμηνεία ακτινογραφίας θώρακα, η διαφωνία μεταξύ αναγνωστών είναι ουσιαστική — στη μελέτη CheXNet, τέσσερις ακτινολόγοι επισήμαναν το ίδιο σύνολο ελέγχου ανίχνευσης πνευμονίας με βαθμολογίες F1 που καλύπτουν περίπου 12 ποσοστιαίες μονάδες {cite}`rajpurkar2017chexnet`, αντικατοπτρίζοντας γνήσια διαγνωστική αβεβαιότητα σε οριακές περιπτώσεις. Για ανίχνευση οζιδίων σε CT πνεύμονα, η ενδο-αναγνωστική μεταβλητότητα (ίδιος αναγνώστης, ίδια περίπτωση, διαφορετική μέρα) μπορεί να είναι τόσο μεγάλη όσο η μεταξύ-αναγνωστική μεταβλητότητα.

Αυτή η μεταβλητότητα δεν είναι απλά θόρυβος — αντικατοπτρίζει γνήσια διαγνωστική αβεβαιότητα. Τα μοντέλα εκπαιδευμένα στους σχολιασμούς ενός μόνο ακτινολόγου μπορεί να μαθαίνουν τις συγκεκριμένες προκαταλήψεις εκείνου του ιατρού αντί για την υποκείμενη παθολογία.

:::{admonition} Η Διαμάχη CheXNet ως Μάθημα HITL
:class: note

Όταν οι Rajpurkar et al. ισχυρίστηκαν ότι το μοντέλο CheXNet τους «ξεπέρασε την απόδοση ακτινολόγου» στην ανίχνευση πνευμονίας, ο ισχυρισμός αμφισβητήθηκε αμέσως από την κοινότητα ακτινολογίας {cite}`yu2022assessing`. Μέρος της διαμάχης αφορούσε το συγκεκριμένο σύνολο ελέγχου και σύγκριση ακτινολόγων. Αλλά ένα βαθύτερο ζήτημα ήταν μεθοδολογικό: η βάση «απόδοσης ακτινολόγου» χρησιμοποιούσε μοναχικούς αναγνώστες υπό χρονική πίεση, ενώ η κλινική ακτινολογία συνήθως περιλαμβάνει διαβούλευση, σύγκριση με προηγούμενες απεικονίσεις και πρόσβαση σε κλινικό πλαίσιο — τίποτα από αυτά δεν είχε το μοντέλο.

Το δίδαγμα δεν είναι ότι το μοντέλο ήταν καλό ή κακό, αλλά ότι **οι συγκρίσεις απόδοσης απαιτούν προσδιορισμό της ρύθμισης HITL**. Ένα μοντέλο που ξεπερνά έναν μοναχικό ακτινολόγο που διαβάζει ψυχρά μπορεί να είναι λιγότερο ακριβές από έναν ακτινολόγο που χρησιμοποιεί την έξοδο του μοντέλου ως δεύτερη γνώμη. Αυτά είναι διαφορετικά συστήματα με διαφορετικούς τρόπους αποτυχίας, και η διαφορετική συνάθροισή τους τα κάνει να διαφέρουν.
:::

:::{admonition} Ήπιες ετικέτες στην ιατρική
:class: important

Αρκετά ιατρικά έργα ΤΝ έχουν μεταβεί σε χρήση **ήπιων ετικετών** που αντικατοπτρίζουν την κατανομή γνωμών εμπειρογνωμόνων αντί μίας «χρυσής» ετικέτας. Μια ακτινογραφία θώρακα επισημασμένη ως 60% πνευμονία / 40% ατελεκτασία από ομάδα ακτινολόγων φέρει περισσότερες πληροφορίες από μια αναγκαστική δυαδική επιλογή. Τα μοντέλα εκπαιδευμένα σε τέτοιες κατανομές εμφανίζουν καλύτερη βαθμονόμηση και πιο κατάλληλη ποσοτικοποίηση αβεβαιότητας — και αυτή η αβεβαιότητα είναι κλινικά σημαντική, καθώς λέει στον κλινικό γιατρό πότε να συμβουλευτεί, όχι μόνο τι σκέφτεται το μοντέλο.
:::

### Ενεργός Μάθηση για Σπάνιες Παθήσεις

Η ενεργός μάθηση είναι ιδιαίτερα πολύτιμη για σπάνιες ασθένειες και σπάνιες παθολογίες, όπου ακόμα και ένα μεγάλο μη-επισημασμένο σύνολο περιέχει μόνο μικρό αριθμό θετικών περιπτώσεων. Η τυπική τυχαία δειγματοληψία θα σπαταλούσε τον χρόνο εμπειρογνωμόνων επισημαίνοντας ως επί το πλείστον αρνητικές περιπτώσεις.

Η ενεργός μάθηση βάσει αβεβαιότητας φυσικά επιλέγει οριακές περιπτώσεις όπου το μοντέλο είναι αβέβαιο — που, για σπάνιες παθήσεις, τείνουν να είναι θετικές περιπτώσεις και οριακά αρνητικά. Αυτό συγκεντρώνει τον χρόνο ειδικού εκεί που είναι πιο πολύτιμος. Ο συνδυασμός ασύμμετρης εκπαίδευσης κλάσης (με `class_weight='balanced'` ή παρόμοιο) και επιλογής βάσει αβεβαιότητας είναι τυπική πρακτική για εργασίες ανίχνευσης σπάνιας παθολογίας.

## Κλινικός Σχολιασμός ΕΦΓ

Τα ηλεκτρονικά αρχεία υγείας (EHRs) περιέχουν τεράστιο πλούτο κλινικού αφηγηματικού κειμένου: σημειώσεις ιατρών, συνοπτικά αρχεία εξόδου, ακτινολογικές εκθέσεις, παθολογικές εκθέσεις. Η εξαγωγή δομημένων πληροφοριών από αυτό το κείμενο απαιτεί ΕΦΓ — και η ΕΦΓ υψηλής ποιότητας απαιτεί επισημασμένα δεδομένα εκπαίδευσης.

**Κοινές εργασίες κλινικού σχολιασμού ΕΦΓ:**
- **Κλινική NER:** Εντοπισμός φαρμάκων, δοσολογιών, διαγνώσεων, διαδικασιών και συμπτωμάτων σε κείμενο
- **Ανίχνευση άρνησης:** «Χωρίς ένδειξη πνευμονίας» vs. «Πνευμονία επιβεβαιωμένη» — μια κρίσιμη διάκριση που είναι εκπληκτικά δύσκολη
- **Χρονική συλλογιστική:** Διάκριση τρεχουσών παθήσεων από ιστορικό («ιστορικό MI, παρουσίασε στηθάγχη»)
- **Απανθρωποποίηση:** Αφαίρεση Προστατευόμενων Πληροφοριών Υγείας (PHI) για επίτρεψη κοινής χρήσης δεδομένων

**Η απανθρωποποίηση PHI** είναι τόσο εργασία σχολιασμού όσο και απαίτηση διακυβέρνησης δεδομένων. Υπό HIPAA (ΗΠΑ) και GDPR (ΕΕ), τα δεδομένα υγείας δεν μπορούν να κοινοποιηθούν χωρίς αφαίρεση ή ανωνυμοποίηση αναγνωριστικών ασθενών. Αυτοματοποιημένα εργαλεία απανθρωποποίησης υπάρχουν αλλά είναι ατελή· η ανθρώπινη ανασκόπηση αυτοματοποιημένων εξόδων είναι τυπική πρακτική, και το προφίλ κινδύνου είναι ασύμμετρο: τα ψευδώς αρνητικά (χαμένα PHI) δημιουργούν νομική έκθεση, καθιστώντας απαραίτητα συντηρητικά κατώφλια.

### i2b2 / n2c2 ως Πρότυπο

Οι πρωτοβουλίες κοινών έργων i2b2 (Πληροφορική για Ενοποίηση Βιολογίας και Κλίνης) και διάδοχος n2c2 (Εθνικές Κλινικές Προκλήσεις NLP) έχουν κυκλοφορήσει μια σειρά συνόλων δεδομένων κλινικής ΕΦΓ επισημασμένων από εμπειρογνώμονες. Αυτά απεικονίζουν τόσο τη δυνατότητα όσο και το κόστος: οι προσπάθειες σχολιασμού συνήθως περιλαμβάνουν ομάδες κλινικών εμπειρογνωμόνων τομέα που εργάζονται για αρκετούς μήνες, σχολιάζοντας εκατοντάδες έγγραφα ανά πρόκληση.

## Ρυθμιστικές Εκτιμήσεις

Η ιατρική ΤΝ υπόκειται σε ρυθμιστική εποπτεία στις περισσότερες δικαιοδοσίες.

**FDA (Ηνωμένες Πολιτείες):** Λογισμικό ΤΝ/ΜΜ ως Ιατρική Συσκευή (SaMD) απαιτεί προ-αγοράς έγκριση ή άδεια. Το Σχέδιο Δράσης ΤΝ/ΜΜ του FDA του 2021 δίνει έμφαση σε **προκαθορισμένα σχέδια ελέγχου αλλαγών** — τεκμηρίωση του πώς θα ενημερώνεται το μοντέλο και πώς αυτές οι ενημερώσεις θα επικυρώνονται πριν από την ανάπτυξη. Ένα μοντέλο που μαθαίνει συνεχώς από κλινική ανατροφοδότηση είναι, στο πλαίσιο αυτό, διαφορετική συσκευή μετά από κάθε ενημέρωση και μπορεί να απαιτεί επανα-επικύρωση.

**Σήμανση CE (Ευρώπη):** Οι ιατρικές συσκευές συμπεριλαμβανομένων των συστημάτων ΤΝ πρέπει να συμμορφώνονται με τον Κανονισμό Ιατρικών Συσκευών (MDR). Ο MDR απαιτεί κλινική αξιολόγηση, επιτήρηση μετά την αγορά και τεκμηρίωση των δεδομένων που χρησιμοποιούνται για εκπαίδευση και επικύρωση.

**Βασική συνέπεια HITL:** Τα ρυθμιστικά πλαίσια απαιτούν σαφή τεκμηρίωση των διαδικασιών σχολιασμού, των προσόντων σχολιαστών, της αξιοπιστίας μεταξύ αξιολογητών και τυχόν αλλαγών στα δεδομένα εκπαίδευσης. Αυτό δεν είναι γραφειοκρατική επιβάρυνση — είναι το ίχνος ελέγχου που επιτρέπει σε έναν κλινικό γιατρό να κατανοήσει ποια δεδομένα εκπαίδευσης παρήγαγαν την τρέχουσα συμπεριφορά του μοντέλου, και απαιτείται νομικά.

## Σχολιασμός Επιστημονικών Δεδομένων

Πέρα από την υγειονομική περίθαλψη, η HITL ML διαδραματίζει αυξανόμενο και υποεκτιμημένο ρόλο στην επιστημονική έρευνα, όπου η πρόκληση σχολιασμού συχνά συνδυάζει εμπειρογνωμοσύνη τομέα με κλίμακα.

### Αστρονομία: Galaxy Zoo

Το Galaxy Zoo {cite}`lintott2008galaxy` crowdsource-αρε τη μορφολογική ταξινόμηση γαλαξιών από τη Sloan Digital Sky Survey σε επιστήμονες-πολίτες. Το πρωτότυπο έργο συγκέντρωσε πάνω από 40 εκατομμύρια ταξινομήσεις από πάνω από 100.000 εθελοντές, αποδεικνύοντας ότι η μεγάλης κλίμακας crowdsourcing επιστημονικής ταξινόμησης εικόνας είναι εφικτή όταν το έργο μπορεί να αποσυντεθεί σε απλές ερωτήσεις αναπαντήσιμες χωρίς εξειδικευμένη εκπαίδευση.

Η εμπειρία Galaxy Zoo παρήγαγε δύο σημαντικά μεθοδολογικά ευρήματα. Πρώτον, η συμφωνία μεταξύ επιστημόνων-πολιτών και επαγγελματιών αστρονόμων ήταν υψηλή για σαφείς περιπτώσεις και συστηματικά αποκλίνει στις οριακές περιπτώσεις — ακριβώς τις περιπτώσεις όπου η διάκριση έχει επιστημονική σημασία. Η λύση δεν ήταν να απορριφθούν τα δεδομένα επιστήμης πολιτών σε οριακές περιπτώσεις αλλά να αντιμετωπιστεί η κατανομή αποκρίσεων εθελοντών ως ήπια ετικέτα που κωδικοποιεί γνήσια μορφολογική ασάφεια.

### Γενωμική: Ταξινόμηση Παθογονικότητας

Ο σχολιασμός γενωμικών παραλλαγών — απόφαση εάν μια παραλλαγή είναι παθογόνος, καλοήθης ή αβέβαιης σημασίας — είναι πρόβλημα ΕΦΓ υψηλών διακυβευμάτων και εμπειρογνώμονα κρίσης. Κλινικές βάσεις δεδομένων παραλλαγών όπως η ClinVar συγκεντρώνουν ερμηνείες εμπειρογνωμόνων από πολλαπλά εργαστήρια, και η διαφωνία μεταξύ εργαστηρίων είναι κοινή.

### Κλίμα και Επιστήμη Γης

Η επισήμανση δορυφορικής απεικόνισης για αλλαγές χρήσης γης, αποδάσωση, έκταση παγετώνων και πορείες καταιγίδων περιλαμβάνει εμπειρογνώμονες τηλεπισκόπησης και ολοένα και περισσότερο πλατφόρμες επιστήμης πολιτών. Η κύρια πρόκληση HITL σε αυτόν τον τομέα είναι χρονική: οι ετικέτες που δημιουργούνται σήμερα μπορεί να γίνουν παρωχημένες καθώς αλλάζει ο κόσμος.

### Νευροεπιστήμη: Connectomics

Η ανακατασκευή νευρωνικών κυκλωμάτων από εικόνες ηλεκτρονικού μικροσκοπίου — connectomics — απαιτεί σχολιασμό σε επίπεδο εικονοστοιχείου μεμβρανών νευρώνων σε τεράστιες στοίβες εικόνων. Το έργο Eyewire το γαμιφίκαρε, εμπλέκοντας δεκάδες χιλιάδες παίκτες στην ιχνηλασία νευρώνων μέσω 3D τόμων εικόνας.

## Διαχείριση Σχολιαστών Ειδικών

Όταν ο σχολιασμός απαιτεί σπάνια εμπειρογνωμοσύνη, οι συνηθισμένες προσεγγίσεις crowdsourcing (Κεφάλαιο 13) δεν εφαρμόζονται.

**Η θεμελιώδης τάση** είναι ότι οι άνθρωποι που μπορούν να παράγουν τους σχολιασμούς υψηλότερης ποιότητας είναι επίσης οι άνθρωποι του οποίου ο χρόνος είναι πιο πολύτιμος και πιο περιορισμένος. Κάθε απόφαση σχεδιασμού σε έναν αγωγό σχολιασμού ειδικών πρέπει να αξιολογείται έναντι του ερωτήματος: αυτό αξιοποιεί τον σπάνιο χρόνο εμπειρογνωμόνων βέλτιστα;

**Αυτό σημαίνει στην πράξη:**

- **Επιθετικός προ-σχολιασμός.** Χρησιμοποιήστε σχολιαστές χαμηλότερης βαθμίδας, αυτοματοποιημένα μοντέλα ή κανονιστικά συστήματα για δημιουργία υποψηφίων που ο ειδικός ανασκοπεί και διορθώνει αντί να δημιουργεί από μηδέν.

- **Σχεδιάστε για προσοχή εμπειρογνωμόνων, όχι απόδοση.** Διεπαφές σχολιασμού βελτιστοποιημένες για υψηλή απόδοση είναι κατάλληλες για crowdsourcing. Ο σχολιασμός ειδικών συχνά επωφελείται από πλουσιότερες διεπαφές: παράθεση με προηγούμενες περιπτώσεις, εύκολη πρόσβαση σε υλικά αναφοράς, πεδία εμπιστοσύνης σχολιασμού και δυνατότητα επισήμανσης περίπτωσης για συζήτηση.

- **Παρακολουθήστε ρητά μεμονωμένα μοτίβα σχολιαστών.** Με μικρό δεξαμενή ειδικών, είναι εφικτό και σημαντικό να παρακολουθείτε τον ρυθμό συμφωνίας κάθε σχολιαστή με την ομάδα, να επισημαίνετε περιπτώσεις που φαίνονται ασυνεπείς με το ιστορικό τους και να τις συζητάτε σε τακτικές συνεδρίες βαθμονόμησης.

- **Ο σχεδιασμός συνεδρίας έχει σημασία.** Ο ιατρικός σχολιασμός είναι γνωστικά απαιτητικός. Στοιχεία από ακτινολογία και παθολογία υποδηλώνουν ότι τα ποσοστά σφαλμάτων αυξάνονται αισθητά μετά από περίπου 90 λεπτά συνεχούς ανάγνωσης.

## Αγωγός Ενεργούς Μάθησης HITL για Ιατρική Απεικόνιση

```{code-cell} python
import numpy as np
from sklearn.linear_model import LogisticRegression
from sklearn.metrics import roc_auc_score
from sklearn.datasets import make_classification
import matplotlib.pyplot as plt

rng = np.random.default_rng(42)

# Simulate a rare-pathology detection task
# 8% positive class (e.g., rare pathology)
X, y = make_classification(
    n_samples=5000, n_features=100,
    n_informative=20, n_redundant=10,
    weights=[0.92, 0.08],
    random_state=42
)
X_train, y_train = X[:4000], y[:4000]
X_test,  y_test  = X[4000:], y[4000:]

print(f"Training set positive prevalence: {y_train.mean():.1%}")

def run_medical_al(strategy, n_initial=50, budget=300):
    labeled = list(rng.choice(len(X_train), n_initial, replace=False))
    unlabeled = [i for i in range(len(X_train)) if i not in labeled]
    aucs = []

    while len(labeled) < n_initial + budget:
        model = LogisticRegression(max_iter=500, class_weight='balanced')
        model.fit(X_train[labeled], y_train[labeled])

        if len(labeled) % 30 == 0:
            preds = model.predict_proba(X_test)[:, 1]
            aucs.append(roc_auc_score(y_test, preds))

        X_pool = X_train[unlabeled]
        if strategy == 'uncertainty' and len(labeled) >= 10:
            probs = model.predict_proba(X_pool)
            entropy = -np.sum(probs * np.log(probs + 1e-9), axis=1)
            q = int(np.argmax(entropy))
        else:
            q = rng.integers(0, len(unlabeled))

        labeled.append(unlabeled.pop(q))

    return np.array(aucs)

# Show how AL preferentially selects positive examples

n_init = 50

entropy = -np.sum(probs * np.log(probs + 1e-9), axis=1)

pos_rate_rnd = y_train[top50_rnd].mean()
```
