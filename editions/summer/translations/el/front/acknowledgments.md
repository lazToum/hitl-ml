# Ευχαριστίες

Κάθε βιβλίο για την ανθρωποκεντρική μηχανική μάθηση οφείλει να ξεκινά με τους ανθρώπους στον δικό του βρόχο.

**Οι σχολιαστές.** Αυτό το εγχειρίδιο μιλά, εν μέρει, για εσάς — για την εργασία σας, την κρίση σας, και το γεγονός ότι το όνομά σας δεν εμφανίζεται σε κανένα δελτίο δεδομένων, εργασία ή δελτίο τύπου για τα μοντέλα που η δουλειά σας κατέστησε δυνατά. Το πεδίο αρχίζει να αντιμετωπίζει αυτό. Έχει δρόμο ακόμα.

**Οι συντελεστές ανοιχτού κώδικα** των οποίων οι βιβλιοθήκες χρησιμοποιούνται σε κάθε παράδειγμα κώδικα: η κοινότητα NumPy (από το 2006), οι συντελεστές του scikit-learn (από το 2007), η ομάδα PyTorch, η κοινότητα Hugging Face, και οι αναρίθμητοι συντηρητές που διορθώνουν σφάλματα τα σαββατοκύριακα χωρίς αποζημίωση ή αναγνώριση. Ο επιστημονικός υπολογισμός είναι χτισμένος πάνω σε αυτή την αόρατη εργασία.

**Οι ερευνητές** που δημοσίευσαν το έργο τους ανοιχτά — σε προεκτυπώσεις arXiv, σε περιοδικά ανοιχτής πρόσβασης και σε αποθετήρια GitHub — καθιστώντας δυνατό για τους ασκούμενους να μαθαίνουν χωρίς θεσμική πρόσβαση. Και οι ερευνητές που αξιολόγησαν εργασίες προσεκτικά και έντιμα — μια μορφή εξειδικευμένου σχολιασμού που η ακαδημία έχει κατορθώσει να μην αναγνωρίζει ως εργασία.

**Οι δάσκαλοι** που κατέστησαν τη γνώση προσβάσιμη: οι bloggers, οι συγγραφείς tutorials, οι χρήστες Stack Overflow που απαντούν, οι παρουσιαστές συνεδρίων που ανέβασαν τις διαφάνειές τους. Η μάθηση δεν συμβαίνει σε απομόνωση, και δεν συμβαίνει χωρίς ανθρώπους πρόθυμους να εξηγούν πράγματα σε αγνώστους.

**Οι επικριτές.** Οι ερευνητές που επεσήμαναν ότι τα σύνολα δεδομένων κωδικοποιούν προκαταλήψεις, ότι τα δημογραφικά χαρακτηριστικά των σχολιαστών διαμορφώνουν τη συμπεριφορά των μοντέλων, ότι οι άνθρωποι στον βρόχο είναι συχνά οι πιο ευάλωτοι συμμέτοχοι στο σύστημα. Αυτό το πεδίο είναι καλύτερο χάρη στην επιμονή σας να θέτετε ενοχλητικά ερωτήματα.
