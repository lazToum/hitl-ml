## 3A.1 Ελαχιστοποίηση Εμπειρικού Κινδύνου

Το κεντρικό πλαίσιο για την επιβλεπόμενη μηχανική μάθηση είναι η ελαχιστοποίηση εμπειρικού κινδύνου (Empirical Risk Minimization - ERM). Δεδομένων των δεδομένων εκπαίδευσης D = {(x_i, y_i)}^n_{i=1}, αναζητούμε τις παραμέτρους θ που ελαχιστοποιούν το:

```
R̂(θ) = (1/n) Σ L(f_θ(x_i), y_i)
```

Όπου L είναι μια συνάρτηση απώλειας (π.χ. διασταυρούμενη εντροπία για ταξινόμηση, μέσο τετραγωνικό σφάλμα για παλινδρόμηση) και f_θ είναι το μοντέλο.

Ο **πραγματικός κίνδυνος** (true risk) που μας ενδιαφέρει είναι:

```
R(θ) = E_{(x,y)~P} [L(f_θ(x), y)]
```

Το χάσμα γενίκευσης (generalization gap) είναι:

```
R(θ) - R̂(θ) ≥ 0
```

Αυτό το χάσμα είναι πάντα μη αρνητικό (ο εμπειρικός κίνδυνος είναι μεροληπτικά αισιόδοξος) και τείνει να αυξάνεται με την πολυπλοκότητα του μοντέλου σε σχέση με το μέγεθος του συνόλου δεδομένων.

## 3A.2 Η Αντιστάθμιση Μεροληψίας-Διακύμανσης

Για ένα μοντέλο παλινδρόμησης, το αναμενόμενο σφάλμα πρόβλεψης αναλύεται ως εξής:

```
E[(y - f̂(x))²] = Bias²(f̂(x)) + Var(f̂(x)) + σ²
                  ─────────────  ──────────────  ──
                  Συστηματικό    Ευαισθησία      Μη αναγώγιμος
                  σφάλμα         στα δεδομένα    θόρυβος
                                 εκπαίδευσης
```

**Μεροληψία (Bias):** Σφάλμα από το μοντέλο που δεν αποτυπώνει την πραγματική σχέση (υποπροσαρμογή).
**Διακύμανση (Variance):** Σφάλμα από την ευαισθησία στις διακυμάνσεις των δεδομένων εκπαίδευσης (υπερπροσαρμογή).
**σ²:** Τυχαία αβεβαιότητα — δεν μπορεί να μειωθεί.

Η αντιστάθμιση μεροληψίας-διακύμανσης (bias-variance tradeoff): η μείωση της μεροληψίας (χρησιμοποιώντας ένα πιο σύνθετο μοντέλο) συνήθως αυξάνει τη διακύμανση (το μοντέλο προσαρμόζεται στον θόρυβο) και αντίστροφα.

**Επίπτωση στον σχεδιασμό HITL:** Τα μοντέλα υψηλής διακύμανσης επωφελούνται περισσότερο από την επανεξέταση ανθρώπου-στον-βρόχο επειδή τα σφάλματά τους είναι απρόβλεπτα — συγκεντρώνονται σε ασυνήθιστες εισόδους χωρίς συστηματικό μοτίβο. Τα μοντέλα υψηλής μεροληψίας επωφελούνται περισσότερο από τη συστηματική συλλογή δεδομένων για τις κλάσεις εισόδων όπου η μεροληψία είναι υψηλότερη.

## 3A.3 Φράγματα Γενίκευσης

**Φράγμα Διάστασης VC (Vapnik & Chervonenkis):**

Για μια κλάση υποθέσεων H με διάσταση VC d_VC, με πιθανότητα 1-δ:

```
R(θ) ≤ R̂(θ) + √(8/n · (d_VC · ln(2n/d_VC) + ln(4/δ)))
```

Το χάσμα γενίκευσης συρρικνώνεται με ρυθμό O(√(d_VC/n)). Τα πιο σύνθετα μοντέλα (υψηλότερη d_VC) χρειάζονται περισσότερα δεδομένα για να κλείσουν το χάσμα.

**PAC-Bayes Φράγμα:**

Ένα πιο σφιχτό φράγμα για στοχαστικούς ταξινομητές:

```
E_{θ~Q}[R(θ)] ≤ E_{θ~Q}[R̂(θ)] + √(KL(Q||P) + ln(n/δ)) / 2n)
```

Όπου P είναι η εκ των προτέρων (prior) και Q είναι η εκ των υστέρων (posterior) κατανομή πάνω στις παραμέτρους.

## 3A.4 Μετατόπιση Κατανομής (Distribution Shift)

### 3A.4.1 Τύποι Μετατόπισης

**Μετατόπιση συνοδών μεταβλητών (Covariate shift):** Το P(x) αλλάζει αλλά το P(y|x) παραμένει το ίδιο.
```
P_train(x) ≠ P_test(x), αλλά P_train(y|x) = P_test(y|x)
```
Παράδειγμα: φίλτρο spam εκπαιδευμένο σε emails του 2020, εφαρμοσμένο το 2025 — το ύφος των emails άλλαξε αλλά το spam παραμένει spam.

**Μετατόπιση ετικετών (Label shift):** Το P(y) αλλάζει αλλά το P(x|y) παραμένει το ίδιο.
```
P_train(y) ≠ P_test(y), αλλά P_train(x|y) = P_test(x|y)
```
Παράδειγμα: το ποσοστό απάτης αλλάζει εποχιακά· η εμφάνιση της απάτης είναι παρόμοια αλλά η βασική συχνότητά της αλλάζει.

**Ολίσθηση εννοίας (Concept drift):** Η σχέση P(y|x) αλλάζει.
```
P_train(y|x) ≠ P_test(y|x)
```
Παράδειγμα: Ένα μοντέλο προσλήψεων εκπαιδευμένο πριν από μια αλλαγή πολιτικής· ο ορισμός του «προσοντούχου» μετατοπίζεται.

**Μπαλαντέρ (Wildcard):** Εντελώς νέος χώρος χαρακτηριστικών.
```
∃x ∈ X_test : p_train(x) ≈ 0
```
Παράδειγμα: ένα μοντέλο όρασης που συναντά ιατρικές εικόνες αφού έχει εκπαιδευτεί σε φυσικές φωτογραφίες.

### 3A.4.2 Ανίχνευση Μετατόπισης Κατανομής

**Μέγιστη Μέση Απόκλιση (Maximum Mean Discrepancy - MMD):**

```
MMD²(P, Q) = ||E_{x~P}[φ(x)] - E_{x~Q}[φ(x)]||²_H
```

Για πυρήνα RBF k(x, x') = exp(-||x-x'||² / 2σ²):

```python
def compute_mmd(X_train, X_test, sigma=1.0):
    """Εκτίμηση MMD μεταξύ των κατανομών εκπαίδευσης και δοκιμής."""
    def rbf_kernel(X, Y):
        dists = cdist(X, Y, 'sqeuclidean')
        return np.exp(-dists / (2 * sigma**2))
    
    K_tt = rbf_kernel(X_train, X_train)
    K_ee = rbf_kernel(X_test, X_test)
    K_te = rbf_kernel(X_train, X_test)
    
    return K_tt.mean() - 2*K_te.mean() + K_ee.mean()
```

**Απλή Προσέγγιση Παρακολούθησης για HITL Παραγωγής:**
```python
class DistributionMonitor:
    def __init__(self, reference_predictions, window_size=1000):
        self.reference = np.array(reference_predictions)
        self.window_size = window_size
        self.recent = deque(maxlen=window_size)
    
    def update(self, new_prediction):
        self.recent.append(new_prediction)
    
    def drift_score(self):
        if len(self.recent) < self.window_size:
            return None
        recent = np.array(self.recent)
        # Τιμή p δοκιμής KS (χαμηλό p = σημαντική ολίσθηση)
        stat, p_value = ks_2samp(self.reference, recent)
        return stat, p_value
```

## 3A.5 Το Πλαίσιο PAC Μάθησης

Η Πιθανώς Προσεγγιστικά Ορθή (Probably Approximately Correct - PAC) μάθηση τυποποιεί το πότε η μάθηση είναι εφικτή.

Μια κλάση εννοιών C είναι PAC μάσιμη εάν υπάρχει ένας αλγόριθμος A τέτοιος ώστε για οποιαδήποτε ε, δ > 0 και οποιαδήποτε κατανομή P πάνω στο X×Y, δεδομένων m ≥ poly(1/ε, 1/δ, n, size(c)) παραδειγμάτων, με πιθανότητα ≥ 1-δ:

```
R(A(D_m)) ≤ ε
```

**Κρίσιμη επίπτωση για HITL από τη θεωρία PAC:** Η μάθηση απαιτεί πεπερασμένα επισημασμένα παραδείγματα ανάλογα του 1/ε. Για πολύ σπάνιες κατηγορίες (χαμηλή εκ των προτέρων πιθανότητα), το δείγμα που απαιτείται για την εκμάθησή τους αυξάνεται ραγδαία. Η ανίχνευση σπάνιων κατηγοριών είναι μια φυσική εφαρμογή HITL: η ανθρώπινη επανεξέταση αβέβαιων περιπτώσεων σε σπάνιες κατηγορίες είναι συχνά ο μόνος πρακτικός τρόπος για την επίτευξη αποδεκτών ποσοστών σφάλματος.

## 3A.6 Ασκήσεις

**Άσκηση 3A.1:** Ένα φίλτρο spam επιτυγχάνει ακρίβεια 99,2% στο σύνολο δοκιμής του, αλλά 96,1% στην εφαρμογή. Το σύνολο δοκιμής προήλθε από το ίδιο σώμα emails με την εκπαίδευση. Προσδιορίστε: (α) ποιος τύπος μετατόπισης κατανομής είναι ο πιο πιθανός· (β) πώς θα τον διαγνώσκετε· (γ) ποια παρέμβαση HITL θα βοηθούσε περισσότερο.

**Άσκηση 3A.2:** Χρησιμοποιώντας την ανάλυση μεροληψίας-διακύμανσης, εξηγήστε γιατί ένα μεγάλο προεκπαιδευμένο γλωσσικό μοντέλο που έχει υποστεί μικρορύθμιση (fine-tuned) σε ένα μικρό σύνολο δεδομένων ειδικού τομέα μπορεί να είναι πιο επικίνδυνο σε μια εφαρμογή HITL από ένα απλούστερο μοντέλο εκπαιδευμένο σε μεγαλύτερο σύνολο δεδομένων του ίδιου τομέα. Ποια παρακολούθηση θα προσθέτατε;

**Άσκηση 3A.3:** Υλοποιήστε μια απλή έκδοση παρακολούθησης μέγιστης μέσης απόκλισης (MMD) σε Python. Χρησιμοποιήστε την για να ανιχνεύσετε πότε μια παρτίδα νέων προβλέψεων έχει διολισθήσει από μια παρτίδα αναφοράς. Ποιο κατώφλι για το MMD θα χρησιμοποιούσατε για να ενεργοποιήσετε έναν ανθρώπινο έλεγχο;

**Άσκηση 3A.4:** Το μη αναγώγιμο σφάλμα σ² στην ανάλυση μεροληψίας-διακύμανσης είναι ανάλογο με την τυχαία αβεβαιότητα. Εξηγήστε με τεχνικούς όρους γιατί η παρέμβαση του ανθρώπου-στον-βρόχο δεν μπορεί να μειώσει αυτή τη συνιστώσα του σφάλματος, και τι θα πρέπει να κάνει ένα σύστημα HITL όταν ανιχνεύει μια περίπτωση με υψηλή τυχαία αβεβαιότητα.
