# Editions Overview

A 13-page LaTeX document introducing all editions of *Human in the Loop: Misunderstood*.

## Files

| File | Description |
|------|-------------|
| `editions_overview.tex` | LaTeX source for the overview document. |
| `editions_overview.pdf` | Compiled PDF (committed as a distributable). |

## Building

```bash
make overview
```

Or manually:

```bash
cd overview
lualatex editions_overview.tex
```
